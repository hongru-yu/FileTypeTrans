## 项目概述

FileTypeTrans 是一个基于 Python 的文件格式转换工具，使用 uv 管理依赖。该工具能够递归遍历用户指定的源目录，将标准文档文件（文本、Word、PDF、Excel、PowerPoint）统一转换为 Markdown 格式，并在 Downloads 文件夹下重建相同的目录结构进行保存。

---

## 开发方法：测试驱动开发 (TDD)

本项目严格遵循 TDD 开发流程：

1. **RED 阶段**: 先编写测试用例
2. **确认失败**: 运行测试确认失败（预期行为）
3. **GREEN 阶段**: 实现代码使测试通过
4. **验证通过**: 运行测试确认通过
5. **REFACTOR 阶段**: 重构优化代码（保持测试通过）

## 开发总结

### 成功经验

1. **TDD 方法的有效性**
   - 先写测试有助于明确需求
   - 测试驱动的代码质量更高
   - 重构时有测试保障

2. **模块化设计**
   - 每个转换器都是独立的
   - 易于添加新的文件类型支持
   - 代码结构清晰，易于维护

3. **完善的测试覆盖**
   - 93% 的代码覆盖率
   - 所有核心功能都有测试
   - 边界情况和错误处理都有测试
